# ChatStoriesApp

This repository contains the Phase 1 MVP for a mobile app focused on:

- Video posts
- Stories
- Likes, comments, sharing
- Fast app launch and media loading

## Project Structure

- `App.js`: Entry point of the app
- `assets/`: Images, icons, backgrounds
- `components/`: Reusable UI components
- `screens/`: Main screens of the app
- `services/`: Business logic (video, stories, interactions)
- `navigation/`: App navigation
- `utils/`: Helper functions
- `config/`: API and Firebase configuration
